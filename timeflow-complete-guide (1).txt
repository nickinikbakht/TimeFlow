## 🔥 Firebase Configuration (REQUIRED)

### Step 1: Create Firebase Project

1. Go to https://console.firebase.google.com/
2. Click "Create a project"
3. Name: "TimeFlow"
4. Disable Google Analytics (optional)
5. Create project

### Step 2: Enable Services

**Enable Authentication:**
1. Click "Authentication" → "Get started"
2. Click "Email/Password"
3. Toggle ON
4. Save

**Enable Realtime Database:**
1. Click "Realtime Database" → "Create Database"
2. Choose location
3. Start in "Test mode"
4. Enable

**Set Security Rules:**
1. In Realtime Database, click "Rules"
2. Paste this:
```json
{
  "rules": {
    "users": {
      "$uid": {
        ".read": "$uid === auth.uid",
        ".write": "$uid === auth.uid"
      }
    }
  }
}
```
3. Publish

### Step 3: Get Your Config

1. Click gear icon ⚙️ → "Project settings"
2. Scroll to "Your apps"
3. Click web icon `</>`
4. Register app: "TimeFlow"
5. **COPY the firebaseConfig object**

### Step 4: Update app.js

Open `app.js` and find this section (lines 8-15):

```javascript
const firebaseConfig = {
    apiKey: "YOUR_API_KEY",
    authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
    databaseURL: "https://YOUR_PROJECT_ID-default-rtdb.firebaseio.com",
    projectId: "YOUR_PROJECT_ID",
    storageBucket: "YOUR_PROJECT_ID.appspot.com",
    messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
    appId: "YOUR_APP_ID"
};
```

**Replace it with YOUR config from Firebase!**

Example of what it should look like:
```javascript
const firebaseConfig = {
    apiKey: "AIzaSyBxxxxxxxxxxxxxxxxxxxxxx",
    authDomain: "timeflow-12345.firebaseapp.com",
    databaseURL: "https://timeflow-12345-default-rtdb.firebaseio.com",
    projectId: "timeflow-12345",
    storageBucket: "timeflow-12345.appspot.com",
    messagingSenderId: "123456789012",
    appId: "1:123456789012:web:abcdef123456"
};
```

## 🎨 Create App Icons

You need TWO icon files for PWA to work:

### Quick Method (Use Online Tool):

1. Go to https://favicon.io/favicon-generator/
2. Or use https://www.favicon-generator.org/
3. Choose:
   - Text: "TF" or "⏰"
   - Background: Purple (#667eea)
   - Font: Bold
4. Generate and download
5. Rename files:
   - `android-chrome-192x192.png` → `icon-192.png`
   - `android-chrome-512x512.png` → `icon-512.png`
6. Put in your `timeflow` folder

### Manual Method:

Use any design tool (Canva, Figma, Photoshop):
1. Create 512x512px canvas
2. Background: Purple gradient (#667eea to #764ba2)
3. Add icon/emoji: ⏰ or TF text
4. Export as PNG: `icon-512.png`
5. Resize to 192x192px and export: `icon-192.png`

## 🚀 Running Your App

### Option 1: Simple (Just Open File)
```bash
1. Double-click index.html
2. Opens in browser
3. Works! (but PWA won't install)
```

### Option 2: Local Server (Recommended)

**Using Python:**
```bash
cd timeflow
python -m http.server 8000
```
Open: http://localhost:8000

**Using Node.js:**
```bash
npm install -g http-server
cd timeflow
http-server
```
Open: http://localhost:8080

### Option 3: Deploy Online (Free)

**Firebase Hosting (Best for Firebase users):**
```bash
npm install -g firebase-tools
firebase login
cd timeflow
firebase init hosting
# Choose your project
# Public directory: . (dot)
# Single page app: No
firebase deploy
```
You'll get: `https://your-project.web.app`

**Netlify (Easiest):**
1. Go to https://app.netlify.com/drop
2. Drag your `timeflow` folder
3. Done! Get instant URL

**Vercel:**
```bash
npm i -g vercel
cd timeflow
vercel
```

**GitHub Pages:**
1. Create GitHub repo
2. Upload files
3. Settings → Pages
4. Deploy from main branch

## 🔧 Troubleshooting

### "Firebase not configured" error
✅ Check you replaced the config in app.js
✅ No "YOUR_API_KEY" placeholders should remain

### Can't sign up/sign in
✅ Enable Email/Password in Firebase Authentication
✅ Check browser console for errors

### Data not saving
✅ Check Firebase security rules are set
✅ Make sure you're signed in
✅ Check browser console

### PWA won't install
✅ Must use HTTPS or localhost
✅ Need both icon files (192 and 512)
✅ manifest.json must be present
✅ Check browser console for manifest errors

### Start time not auto-filling
✅ You need at least one journal entry first
✅ Last entry must have an end time
✅ Refresh the page after adding entries

### Icons not showing
✅ Make sure icon-192.png and icon-512.png exist
✅ Check file names are exactly correct
✅ Files must be in same folder as index.html
